# OREO — Twist. Lick. Dunk.

**Description:** A premium, single-page brand landing site for OREO — built from scratch in pure HTML/CSS/JS with no frameworks. Dark, editorial-style design (Bebas Neue + Playfair Display + DM Sans) with a gold-on-black aesthetic, custom cursor, and rich scroll-driven motion throughout.

**Highlights:**
- 🖱️ Custom dual-layer cursor (dot + trailing ring) replacing the system pointer
- 🎬 Scroll-reveal animations via `IntersectionObserver` — staggered fade/slide-up on every section
- 🍪 Hand-built SVG cookie illustrations (hero background, hero visual) with slow rotation and floating motion
- 🛍️ 12-flavor product catalog (Original, Double Stuf, Golden, Mint, Birthday Cake, Peanut Butter, Red Velvet, S'mores, Thins, Dark Chocolate, Salted Caramel Thins, Irish Crème Thins) with live category filtering (All / Classic / Flavored / Thins / Limited)
- 🛒 Add-to-cart micro-interaction with a sliding cart indicator and live count
- 🎞️ Infinite CSS marquee strip cycling through every flavor name
- 💬 Testimonials grid, newsletter signup, and a "Buy 3 Get 1 Free" promo block
- 📱 Responsive breakpoints down to mobile (grid collapses, decorative visuals hide gracefully)

**Tech Stack:** Vanilla HTML, CSS (custom properties, keyframe animations, CSS grid), JavaScript (no dependencies) — Google Fonts (Bebas Neue, Playfair Display, DM Sans).

**Status:** Complete, deployed as a static single-file site.
