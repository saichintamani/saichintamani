# PrepMind AI

**Description:** An EdTech SaaS platform built for Indian engineering students preparing for campus placements — combining structured prep content with AI-assisted practice.

**Tech Stack:**
- Frontend: React + Vite
- Backend: FastAPI
- Database/Auth: Supabase + Clerk
- Payments: Razorpay
- Automation: n8n
- AI: OpenAI GPT-4o

**Brand Identity:** Orange (`#FF6B00`) as the core brand color.

**Status:** In active development.
